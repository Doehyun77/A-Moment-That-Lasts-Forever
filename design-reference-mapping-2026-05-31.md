## 현재 화면 인상 요약
현재 첫 화면은 웨딩 소개형 랜딩이라기보다 **관리자 로그인 진입 화면** 인상이 강하다.
핵심 개선 포인트는 다음 5가지다.

1. 첫 화면 정보 우선순위 재정리
2. CTA 버튼 위계 정리
3. 카드/섹션 구획감 강화
4. 갤러리/안내/FAQ 같은 본문 섹션의 존재감 강화
5. 타이포와 컬러를 더 정돈된 웨딩 톤으로 통일

---

## 1:1 매핑표

### 1) 현재: 첫 화면 중앙 로그인 카드
- 추천 레퍼런스
  - **Lazyweb**: 정보 우선순위, 히어로 카피 + CTA 2개 구조
  - **Material 3**: 표면 위계, 라운드, CTA 강조 규칙
- 바꿀 방향
  - 로그인 카드를 단독 주인공처럼 보이게 하지 말고
  - 상단에 브랜드/웨딩 안내 히어로를 먼저 두고
  - 로그인은 운영자 전용 진입점으로 존재감을 낮춘다
- 적용 포인트
  - 메인 헤드라인 1개
  - 보조 설명 1~2줄
  - 주 CTA 1개 + 보조 CTA 1개
  - 로그인은 하단 또는 별도 관리자 링크로 이동

### 2) 현재: CTA 버튼(로그인 버튼 위주)
- 추천 레퍼런스
  - **shadcn/ui**: pill형/outline형 버튼 구조
  - **Material 3**: primary / secondary 위계
- 바꿀 방향
  - 단일 브라운 버튼만 강하게 두는 대신
  - 사용자용 CTA와 관리자용 CTA를 분리
- 적용 포인트
  - Primary: `청첩장 보기` 또는 `오시는 길`
  - Secondary: `축하 메시지 남기기` 또는 `FAQ 보기`
  - Admin: 텍스트 링크 또는 작은 outline 버튼

### 3) 현재: 상단 네비게이션/헤더 부재
- 추천 레퍼런스
  - **Lazyweb**: 간결한 헤더 + 우측 CTA
  - **oh-my-design**: 섹션 시작부의 또렷한 구획감
- 바꿀 방향
  - 상단에 너무 많은 메뉴를 두지 말고
  - `청첩장`, `웨딩사진`, `FAQ`, `관리자` 정도만 남긴다
- 적용 포인트
  - 좌측: 로고/브랜드명
  - 중앙 또는 좌측: 3~4개 메뉴
  - 우측: `오시는 길` 또는 `참석 의사 전달`

### 4) 현재: 본문 섹션 구획감 약함
- 추천 레퍼런스
  - **oh-my-design**: 큰 섹션 간격과 배경 전환
  - **Material 3**: 선보다 면으로 구분하는 표면 계층
- 바꿀 방향
  - 긴 한 화면 흐름보다
  - 구역별로 성격이 보이게 분리
- 적용 포인트
  - 히어로
  - 예식 정보
  - 갤러리
  - FAQ
  - 축하 메시지/방명록
  - 오시는 길
  각 섹션 사이 간격을 키우고 배경톤을 미세하게 바꾼다.

### 5) 현재: 갤러리/사진 섹션의 존재감 부족
- 추천 레퍼런스
  - **shadcn/ui**: carousel / card 구조
  - **Dribbble wedding website**: 감성 사진 중심 레이아웃
- 바꿀 방향
  - 사진이 단순 부속이 아니라 사이트 감정선을 만드는 핵심이 되게 한다
- 적용 포인트
  - 대표 사진 1장 크게
  - 하단 썸네일 3~6장
  - 필요시 `사진 더 보기` CTA 배치

### 6) 현재: FAQ/안내 정보가 버튼 수준으로만 보임
- 추천 레퍼런스
  - **shadcn/ui**: accordion
  - **Material 3**: 카드 그룹과 본문 위계
- 바꿀 방향
  - FAQ를 단순 버튼이 아니라
  - 실제 질문/답변 묶음으로 정리
- 적용 포인트
  - `주차는 어디에 하나요?`
  - `식 시작 시간은 언제인가요?`
  - `아이 동반 가능한가요?`
  - `축의금/마음 전하실 곳`
  이런 질문형 아코디언으로 구성

### 7) 현재: 전반적 타이포가 템플릿 느낌
- 추천 레퍼런스
  - **Dribbble wedding website**: 세리프 헤드라인 + 정돈된 본문
  - **Material 3**: 제목/본문/보조 텍스트 위계
- 바꿀 방향
  - 제목만 웨딩 감성 세리프로
  - 본문은 읽기 쉬운 산세리프로 유지
- 적용 포인트
  - H1/H2만 감성 포인트
  - 본문/설명/메타는 단정하게 유지

### 8) 현재: 전체 톤이 예쁜데 약간 정적이고 올드하게 느껴질 수 있음
- 추천 레퍼런스
  - **Dribbble**: 저채도 웨딩 팔레트
  - **Material 3**: 색상 위계 체계
- 바꿀 방향
  - 플로럴 장식은 유지하되
  - 배경, 카드, 버튼, 텍스트의 역할을 더 명확히 한다
- 적용 포인트
  - 배경: 아이보리
  - 카드: 화이트/크림
  - 텍스트: 차콜
  - 포인트: 로즈베이지 또는 세이지 1개
